Automation system for educational institutions
## How to install:  
- Install `Python 3.9` and `pipenv`
- Install requirements `pipenv install`
- Edit config file:
  - Run project to create `settings.conf`
  - Edit `settings.conf`
- Migrate database `python manage.py migrate`
